Book 1:1 mock interviews with senior engineers. Get AI-powered feedback. Land your dream job.

- 🐳 **Custom Docker execution engine** — sandboxed code runner I built from scratch. Each submission spins up an isolated container with memory, CPU & network limits. Same architecture as LeetCode.
- 💻 **Real-time collaborative code editor** — synced between interviewer & candidate via WebSockets with echo-loop prevention
- 🤖 **AI question co-pilot** — generates role-specific interview questions on demand using Gemini
- 🎥 **HD video calls** with screen sharing, recording, and live chat
